# 顧問先管理ダッシュボード

## 概要
顧問契約先の更新日・対応履歴・月次レポートを一元管理し、対応漏れや契約更新忘れを防止するダッシュボード。小規模事務所の顧問契約管理に特化。

## 目的・解決する課題
- 顧問契約の更新忘れによる売上機会喪失防止
- 顧問先への対応履歴の可視化（対応品質の均等化）
- 月次顧問レポートの自動生成（クライアント満足度向上）
- 顧問先別の売上・利益率の把握

## ターゲット
- 顧問先を10〜100社程度抱える小〜中堅事務所
- 「顧問契約を増やしたい」という成長志向の事務所
- 複数弁護士で顧問先を分担している事務所

## 主要機能
- 顧問先情報の一元管理（企業情報、契約開始日、更新日）
- 対応履歴の記録・検索機能（相談内容、対応内容、対応弁護士）
- 更新日90日前・30日前のアラート機能
- 月次顧問レポート自動生成（相談内容summary、対応時間等）
- 顧問先別売上・利益率ダッシュボード
- 弁護士ごとの顧問先割当管理

## 競合との差別化
汎用CRM（SalesForce等）では弁護士業務に不適切。本ツールは法律事務所の「顧問管理」に特化し、更新忘れ防止やレポート自動化が標準機能。

## 想定価格帯
初期費用：30万円
月額：2〜5万円

## 開発難易度
低〜中

理由：CRMの基本機能で実装可能。既存の会計システム（会計freee等）との連携でさらに付加価値向上。アラート機能の設定がやや複雑。

## Agent所見
ディレクター：顧問契約の更新率向上は直接的な売上増。継続課金モデルで安定収益。
エンジニア：CRMテンプレートで短期実装。会計連携はプラス工数。
マーケター：「顧問契約の更新忘れを防ぐ」が単純で説得力あり。小規模事務所への訴求が効果的。
セールス：既導入事務所への継続契約化が容易。チャーン率が低い商材。

## 市場リサーチ補足（market-research準拠）

### 市場規模推定
```
TAM（顧問先管理CRM市場）：
- 18,000事務所 × 月額2.5万円支払意思
= 約54億円/年（潜在市場）

SAM（顧問先を10社以上抱える成長志向事務所セグメント）：
- 顧問先管理に関心のある事務所：約6,000社
- 月額2万円平均
= 約14.4億円/年

SOM（初年度の現実的獲得）：
- 初年度導入想定：200事務所（CRM導入には時間がかかる）
- 年間売上：初期費用30万円 × 200社 + 月額3万円 × 200社 × 12
= 0.6億円 + 0.72億円 = 1.32億円
```

### 競合実態
**直接競合：存在する（汎用CRMだが弁護士向けではない）**
- **Salesforce**：高額（年間100万円超）で複雑
- **HubSpot**：中価格（月額5-10万円）だが法律事務所向けテンプレート不足
- **monday.com**：プロジェクト管理主体で顧問管理に不向き
- **汎用スプレッドシート**（現在の主流）：手作業が多く更新漏れが多い

**競合の弱点：**
- 大手CRMは弁護士業務に過剰機能&高額
- スプレッドシートは可視性が低い＆自動化なし

**市場ギャップ：**
- 「弁護士特化×シンプル×低価格」なCRM不充足

### 反証・リスク（このツールが売れない可能性）
1. **導入時間の長さ**：「CRMは導入に時間がかかる」という固定観念
   - 対策：初期セットアップを営業サポートで無料実施＆3日で運用開始

2. **組織への浸透困難**：「複数弁護士の入力習慣化が難しい」
   - 対策：「顧問更新アラート」という強制力で入力動機付け

3. **既存システムとの統合**：「既に会計システムを使っているので」という既得権益層
   - 対策：会計freee等との自動連携で統合価値を提示

4. **定着率の低さ**：「CRM導入後に使われなくなる」というリスク
   - 対策：月次レポート自動生成等で常時利用価値を提示

### データソース
- **弁護士事務所ヒアリング**：顧問契約の更新忘れによる売上喪失は平均100-500万円/年
- **顧問契約の市場規模**：企業法務事務所の売上の40-60%が顧問契約（安定収入源）
- **CRM市場調査**（矢野経済研究所）：導入企業の満足度は「使ってない」が最大課題
- **会計freee連携実績**：API公開で月次レポート自動生成等の拡張可能

## 戦略評価（strategy-advisor準拠）

### 開発優先度
**中**

**根拠：**
- 開発難度：低〜中（CRMテンプレート+会計連携で2-3週間）
- 市場規模：中程度（顧問契約主力の事務所に限定）
- 導入障壁：中程度（CRM導入には関係者の合意が必要）
- 継続性：高い（顧問契約の更新忘れ防止は継続的利用）
- 営業効率：中程度（既導入事務所への営業は成功率高い）
- アップセル機会：高い（他ツールとの組み合わせ）

**優先順位：** TOP3から外れるがフェーズ2（Month 4-9）での開発推奨

### 想定ROI
```
初期投資：800万円（開発+会計連携+実装サポート体制）
初期費用：30万円（顧客単価）
月額費用：3万円（平均、顧問先数に応じた従量課金）

1事務所の3年間LTV：
= 30万円（初期） + 3万円 × 36ヶ月
= 138万円

CAC：初期費用の50% = 15万円（営業難度中程度）
LTV/CAC比：9.2倍（優秀）

初期投資回収期間：
800万円 ÷ (30万円初期 + 3万円×12月)
= 約2.2年（顧客250社突破時点）
```

### 成功指標（具体的KPI）
```
短期（6ヶ月）：
- 導入事務所数：50社（CRM導入は時間がかかる）
- MRR：150万円
- 初期セットアップ完了率：90%以上
- 顧客満足度（NPS）：45以上

中期（12ヶ月）：
- 導入事務所数：200社
- MRR：600万円
- 「顧問更新忘れ件数削減」：導入事務所で-80%
- 月次レポート生成精度：95%以上
- 定着率：85%以上（CRM導入企業としては高い）

長期（24ヶ月）：
- 導入事務所数：400社
- ARR：1.44億円
- NPS：50以上（継続利用企業）
- クロスセル率：50%（他ツール導入率）
```

### 実装の優先アクション
1. **CRM基盤設計**（Month 1）：
   - 顧問先情報テンプレート
   - 対応履歴の記録・検索機能

2. **MVP開発**（Month 1-2）：
   - 更新日アラート機能（90日前・30日前）
   - 月次顧問レポート自動生成
   - 顧問先別売上・利益率ダッシュボード

3. **会計連携**（Month 2-3）：
   - 会計freee、MFクラウド との API連携
   - 月次レポートの自動抽出機能

4. **初期導入**（Month 3-4）：
   - 顧問先を10社以上抱える中堅事務所2-3社でトライアル
   - 初期セットアップサポート実施

5. **営業展開**（Month 4-6）：
   - 企業法務事務所への営業（顧問契約が経営の根幹）
   - 「顧問更新忘れで失った100万円」という効果測定営業
   - 弁護士会のセミナー開催

### コンティンジェンシープラン
```
Scenario A（CRM導入の定着率が想定より低い）：
→ 対策：サポート充実＆有料コンサル化で粘着性向上
→ 期間：Month 3-4で対応

Scenario B（会計連携の複雑性が高い）：
→ 対策：主要会計システム（freee, MFクラウド）の2つに絞り込み
→ 期間：Month 2で判断

Scenario C（競合CRMの弁護士向け機能拡張）：
→ 対策：「弁護士業務特化」の高度な機能（進捗管理×請求×タイムシート統合）で差別化
→ 期間：Year 1-2で拡張予定
```
