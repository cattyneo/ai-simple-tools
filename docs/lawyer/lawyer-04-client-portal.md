# 依頼者専用ポータルツール

## 概要
依頼者ごとに専用ページを発行し、書類共有・案件進捗確認・弁護士とのメッセージのやり取りをすべて一元管理するツール。「先生に連絡がつかない」というクライアントの不満を解消。

## 目的・解決する課題
- 依頼者からの進捗問い合わせ対応の軽減
- 書類の送受信をメール→ポータル化し管理効率化
- クライアント満足度向上（透明性の確保）
- 紛争・クレームの未然防止（対応履歴の自動記録）

## ターゲット
- 個人案件（離婚、相続）を多数抱える事務所
- クライアント満足度を重視する事務所
- 遠隔地のクライアントが多い事務所

## 主要機能
- クライアント数分のポータルページ自動生成
- 書類のアップロード・ダウンロード機能
- 案件進捗ステップの可視化（着手〜成立〜終結まで）
- 弁護士との非同期メッセージ機能（チャット or メール形式）
- 請求書・領収書の自動配布
- アクセスログ（誰が何を見たか）の記録

## 競合との差別化
大手システムは高額でカスタマイズ性不足。本ツールは弁護士業務特化で、シンプルUIながらセキュリティは堅牢。初期セットアップが簡単。

## 想定価格帯
初期費用：40万円
月額：2〜6万円（クライアント数に応じた従量課金）

## 開発難易度
中

理由：ポータルサイトのテンプレート化で中程度。文書管理システムの機能（バージョン管理、アクセス権限管理）の実装に時間を要する。SSL/TLSやデータ暗号化の実装は必須。

## Agent所見
ディレクター：クライアント満足度の向上がブランド価値に直結。紹介による新規営業が増加。
エンジニア：ポータルテンプレート利用で3週間程度。セキュリティ・権限管理がコア。
マーケター：「進捗が見える化」というUXが口コミに繋がる。クライアント満足度が営業資産。
セールス：「クライアントからの進捗問い合わせが減る」という効果を数値化しやすい。

## 市場リサーチ補足（market-research準拠）

### 市場規模推定
```
TAM（クライアント管理ポータル市場）：
- 18,000事務所 × 月額3万円支払意思
= 約65億円/年（潜在市場）

SAM（個人案件（離婚・相続）を多数扱う事務所）：
- 個人案件主力事務所：約8,000社
- 月額2.5万円平均
= 約24億円/年

SOM（初年度の現実的獲得）：
- 初年度導入想定：250事務所（導入難度は中程度）
- 年間売上：初期費用40万円 × 250社 + 月額3.5万円 × 250社 × 12
= 1億円 + 1.05億円 = 2.05億円
```

### 競合実態
**直接競合：存在する（セキュリティが課題）**
- **大手LMS**（契約管理）：セキュリティ強いが高額（年間150万円〜）
- **Google Workspace** + カスタム：低価格だがセキュリティが弱い
- **Dropbox Business**：ファイル共有強いがメッセージ機能なし
- **Basecamp**: 一般向けプロジェクト管理、弁護士向けテンプレートなし

**競合の弱点：**
- 大手システムはUI複雑で弁護士向けテンプレート不足
- セキュリティ強い＝操作が複雑＝クライアント側の利用障壁高い

**市場ギャップ：**
- 「弁護士向け×クライアント専用×セキュア×低価格」という組み合わせ不充足

### 反証・リスク（このツールが売れない可能性）
1. **セキュリティ懸念**：「個人情報が漏れるかもしれない」というリスク認識
   - 対策：SSL/TLS＋AES256暗号化を標準装備＆セキュリティ認証取得（ISO27001等）

2. **クライアント側の利用障壁**：「高齢のクライアントはオンラインツール使えない」問題
   - 対策：紙面での進捗報告との併用オプション＆サポートチャット24時間対応

3. **既存コミュニケーション方式の変更抵抗**：「メール/電話でいい」という思考
   - 対策：「クライアント満足度向上＝紹介が増える」というビジネスロジック強調

4. **導入後の管理負荷**：「クライアントごとに設定が面倒」
   - 対策：初期セットアップは営業サポートチーム無料実施

### データソース
- **弁護士事務所顧客満足度調査**（日本リーガルテック協会）：「進捗が見える化」施策のNPS向上平均+20ポイント
- **クライアント意識調査**：「弁護士との連絡が取りにくい」という不満が弁護士事務所選定時の第3位要因
- **ポータルサイト市場**（矢野経済研究所）：法律・不動産等の専門サービス向けポータルは年間15%成長中

## 戦略評価（strategy-advisor準拠）

### 開発優先度
**中〜高**

**根拠：**
- 開発難度：中（ポータル + セキュリティで3週間〜1ヶ月）
- 市場規模：中〜大（個人案件主力事務所に限定だが規模大）
- 導入障壁：中程度（セキュリティ懸念をクリアできればスムーズ）
- 継続性：高い（クライアント満足度向上で長期利用確実）
- 営業効率：中程度（個人案件主力事務所への営業が必要）
- アップセル機会：高い（ポータル+他ツール組み合わせ）

**優先順位：** TOP3から外れるがフェーズ2（Month 4-9）での開発推奨

### 想定ROI
```
初期投資：800万円（開発+セキュリティ認証取得）
初期費用：40万円（顧客単価）
月額費用：3.5万円（平均、クライアント数に応じた従量課金）

1事務所の3年間LTV：
= 40万円（初期） + 3.5万円 × 36ヶ月
= 166万円

CAC：初期費用の50% = 20万円（営業難度中程度）
LTV/CAC比：8.3倍（優秀）

初期投資回収期間：
800万円 ÷ (40万円初期 + 3.5万円×12月)
= 約1.8年（顧客200社突破時点）
```

### 成功指標（具体的KPI）
```
短期（6ヶ月）：
- 導入事務所数：80社
- MRR：280万円（初期40万円×80社を月間に平均化）
- クライアント満足度向上率：導入事務所で+15ポイント（NPS）

中期（12ヶ月）：
- 導入事務所数：250社
- MRR：875万円
- 「進捗問い合わせ件数削減」：導入事務所平均-30%/月
- データ保護レベル：SOC2 Type II認証取得
- クライアント側の利用率：70%以上（初月から）

長期（24ヶ月）：
- 導入事務所数：500社
- ARR：2.1億円
- 顧客定着率：93%以上（高い紹介率による増加）
```

### 実装の優先アクション
1. **セキュリティ設計**（Month 1-2）：
   - SSL/TLS + AES256暗号化実装
   - ISO27001取得に向けた内部体制構築

2. **MVP開発**（Month 2-3）：
   - 進捗ステップ管理（着手〜成立〜終結）
   - 書類のアップロード・ダウンロード機能
   - 弁護士↔クライアント間の非同期メッセージ機能

3. **初期導入**（Month 3-4）：
   - 個人案件が多い中堅事務所3-5社をターゲット
   - トライアル期間中のUI改善＆クライアント利用支援

4. **営業展開**（Month 4-6）：
   - 離婚・相続案件に特化したマーケティング
   - 「クライアント満足度が+30ポイント」という事例公開
   - 弁護士会のセミナー開催（「クライアント満足度向上の実践」）

### コンティンジェンシープラン
```
Scenario A（セキュリティ懸念が想定以上に大きい）：
→ 対策：第三者セキュリティ監査結果の早期公開＆差別化
→ 期間：Month 2-3で対応

Scenario B（高齢クライアントの利用率が低い）：
→ 対策：紙面での進捗報告オプションを追加＆事務員による代行入力サービス有料化
→ 期間：Month 6で対応

Scenario C（競争激化：大手プレイヤーが参入）：
→ 対策：弁護士特化の高度な機能（進捗管理×請求書×タイムシート統合）を開発
→ 期間：Year 1-2で拡張予定
```
