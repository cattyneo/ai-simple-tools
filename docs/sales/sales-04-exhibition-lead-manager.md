# 展示会・セミナーリード一括管理ツール

## 概要
展示会・セミナーで回収した名刺を自動スキャン→データ化→AI ランク付け→フォローメール自動送信まで、ワンストップで管理。「名刺の山」から即座にアクションへ。

## 目的・解決する課題
- 展示会後、回収した名刺のデータ入力に営業が膨大な時間を費やしている
- リード優先度の判定が属人的で、フォロー漏れが発生している
- フォローメール送信のタイミング・内容が場当たり的

## ターゲット
展示会・セミナーに年3回以上出展する、営業組織30名以上の BtoB企業。イベント営業が多い業界（SaaS、コンサルティング、IT ソリューション等）。

## 主要機能
- スマートフォンカメラで名刺を撮影→自動 OCR データ化（名前・会社・役職・連絡先の抽出）
- 社内 CRM 照合（既存顧客か新規リードか自動判定）
- AI による質問対話（「課題は？」「予算時期は？」）から自動ランク付け（S/A/B/C等）
- リードランク別に自動フォローメール送信
- 同一企業の複数名刺を自動で組織化
- チーム全体のリード進捗ダッシュボード

## 競合との差別化
名刺スキャン SaaS（Sansan・Eight等）は存在するが、AI ランク付け + フォローメール自動送信の機能は充実していない。BowNow等の無料 MA ツールとの組み合わせで対応している企業も多いが、本ツールは「展示会リード特化」でワンストップ化。

## 想定価格帯
初期費用: 40万円 / 月額: 1.5～3万円（スキャン名刺数に応じた従量課金）

## 開発難易度
低～中 / OCR（Tesseract、Google Vision API等）+ AI ランク付けロジック（ルールベース + LLM 質問生成）で実装可能。展示会 UI/UX の直感性がキー。

## Agent所見
- **ディレクター**: イベント営業の生産性向上が具体的に可視化でき、経営層へのプレゼンが容易。展示会企業の複数参加実績をケーススタディ化することで信頼感が醸成される。
- **エンジニア**: OCR + LLM の組み合わせは実装難易度は低め。Sansan 等との API 連携オプションを検討すれば、ユースケースが広がる。
- **マーケター**: イベント後の「営業の疲弊感」を解消する訴求が刺さる。「展示会後 48 時間以内に最初のメール送信」といった数値化された利点を強調。
- **セールス**: 展示会出展企業は「リード化率」を重視しているため、本ツールの「自動フォロー機能による反応率向上」を数値で提示すれば受注確度は高い。複数年契約での割引提案が有効。

## 市場リサーチ補足（market-research準拠）

### 市場規模推定
- **TAM（Total Addressable Market）**: 展示会出展企業（日本国内年3回以上出展 ≈ 3,000～5,000社） × 月額1.5～3万円 × 12か月 ≈ 54～180億円
- **SAM（Serviceable Addressable Market）**: 展示会出展企業で営業組織30名以上 ≈ 800～1,500社 ≈ 約14.4～54億円
- **SOM（Serviceable Obtainable Market）**: 初期3年で狙える市場 ≈ 150～300社 × 年単価36万円 ≈ 5.4～10.8億円

### 競合実態
- **Sansan**: 名刺スキャン特化。業界トップシェア。月額3～10万円（高額）
- **Eight**: 名刺スキャン + 簡易ランク付け。個人向けメインで企業向けは限定的
- **BowNow**: 無料MA。展示会リード管理と組み合わせで対応。ただしAI自動ランク付けなし
- **本ツールの差別化**: 「展示会リード特化」「AI自動ランク付け」「フォローメール自動送信」の統合機能。Sansanより廉価。

### 反証・リスク要因（このツールが売れない可能性）
- **Sansanの追い上げ**: Sansanが廉価ティア（月1～2万円）を投入した場合、本ツール不要化の可能性
- **OCR精度の問題**: 手書き名刺や複数言語対応が必要な場合、認識精度が50%以下に低下する可能性
- **展示会市場の景気変動**: 経済悪化によって企業の展示会出展数が減少する可能性
- **リード質の問題**: 展示会で回収した名刺の「購買意欲」が想定以上に低い場合、ランク付けアルゴリズムが機能しない
- **フォローメール開封率の低下**: 展示会後のフォローメール開封率が業界平均20%以下で、自動送信の効果が限定的

## 戦略評価（strategy-advisor準拠）

### 開発優先度
- **総合判定: 高**
- **理由**: 展示会市場のマクロトレンド（コロナ後回復）と、ユーザーペイン（名刺の山からのリード化）が明確に一致。市場進出タイミングが絶好。Sansanに対する差別化（廉価×AI自動ランク付け）が明確。

### 想定ROI（初期投資回収期間）
- **初期投資**: 開発費400万 + 営業・マーケティング初期費用200万 = 600万円
- **月額平均客単価**: 2.5万円（年間30万円）
- **想定初期採用速度**: 6か月で15社、12か月で40社
- **初期12か月MRR**: 15社×2.5万円=37.5万円 → 40社×2.5万円=100万円
- **回収期間**: 初期12か月で約80万円×12か月 = 960万円 → 8か月程度で回収
- **ROI指標**: 初期投資600万 / 3年LTV（40社×2.5万円×36か月）= 3,600万 → **ROI: 6倍（3年ベース）**

### 成功指標（KPI）
- **短期（6か月）**: ベータテスト企業8社からの「展示会後リード化率向上」30%以上の実績報告
- **中期（12か月）**:
  - 有料顧客: 40社
  - MRR: 100万円
  - 顧客の「リード化率向上」: 平均25%以上改善
  - ユーザーNPS: 60以上
- **長期（24か月）**:
  - 有料顧客: 100社
  - MRR: 250万円
  - 顧客紹介による新規リード: 全新規リードの40%以上
  - Sansan乗り換え顧客率: 25%以上

### コンティンジェンシープラン（リスク対応）
**リスク1: Sansanが廉価ティアを投入した場合**
- 対応: 「展示会リード特化」「AI自動ランク付けの精度」での差別化強調。顧客成功プログラムの充実（3か月ごとのビジネスレビュー等）でロックイン効果向上。

**リスク2: OCR精度が期待値を下回る場合**
- 対応: 複数OCRエンジン（Google Vision API + Azure + Tesseract）を並列利用。ユーザーによる手動修正UI充実化。ユーザーフィードバック基に機械学習モデル継続改善。

**リスク3: 展示会出展企業の新規採用速度が期待値の50%以下の場合**
- 対応: 展示会運営企業（日本展示会協会等）とのパートナーシップ構築。展示会企業への「無料トライアル」プログラム拡充。
