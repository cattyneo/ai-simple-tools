# BtoB 向け SNS 投稿スケジューラー×効果測定ツール

## 概要
X（Twitter）・LinkedIn・note への投稿を一元管理し、スケジュール配信。エンゲージメント（いいね・リプライ・シェア）をトラッキング。BtoB 企業の SNS 運用に特化し、Hootsuite などの BtoC 向けツールとの差別化を実現。

## 目的・解決する課題
- 複数の SNS 運用が負担になり、一貫性のある投稿ができていない
- SNS 投稿の効果測定（エンゲージメント→リード化率）が不明確
- リード生成ツール（Hootsuite・Buffer等）は BtoC 向けで、BtoB に適していない

## ターゲット
BtoB 企業の情報発信・マーケティング部門。LinkedIn・X をビジネス発展に活用したいが、運用工数が足りていない層。CEO・営業推進者が個人発信も含めて統一管理したいニーズもある。

## 主要機能
- X・LinkedIn・note への投稿を一画面で管理・スケジュール配信
- 業界別・コンテンツタイプ別のテンプレート投稿提案機能
- 投稿のエンゲージメント（いいね・リプライ・シェア・クリック）をリアルタイム集計
- エンゲージメント の多い投稿パターンを AI が自動分析・提案
- リード化追跡（LinkedIn の プロフィール訪問数、招待リクエスト増加等）を計測
- チームメンバー用アカウントで、投稿スケジュール・承認フロー管理
- 業種別・フェーズ別のベストプラクティス投稿ライブラリ
- Google Analytics との連携で、SNS → Web サイト への流入追跡

## 競合との差別化
Hootsuite・Buffer は BtoC 向けで、LinkedIn 機能が不十分。HubSpot 等の MA ツールは SNS 機能が付随的。本ツールは「BtoB 向けの SNS 運用」に特化し、X・LinkedIn・note の最適化を実現。CEO・営業個人の発信を組織的に管理するユースケースも未開拓。

## 想定価格帯
初期費用: 25万円 / 月額: 2～3.5万円（メンバー数に応じた従量課金）

## 開発難易度
中 / Twitter API v2・LinkedIn API・note API との連携 + エンゲージメント計測ロジック + GA4 との連携で実装可能。各 SNS の API 仕様変更への対応がメンテナンスコスト。

## Agent所見
- **ディレクター**: BtoB 企業の「SNS による認知拡大 → リード獲得」というファネルを設計できる点が強み。ただし効果測定（SNS → 商談への帰属）が複雑なため、初期導入時の期待値調整が必須。
- **エンジニア**: 複数 SNS API の連携が実装のコア。各 API の認証・レート制限への対応は細かいが、対応難易度は高くない。GA4 統合で「SNS が本当にリード源になっているか」の可視化ができると差別化になる。
- **マーケター**: BtoB 企業の情報発信をスケーラブルにできる点が訴求のメイン。「個人の属人的な SNS 運用から組織的な情報発信へ」というスピン化が有効。業界別テンプレート投稿の充実度が競争力。
- **セールス**: LinkedIn ユーザー企業をターゲットにすることで、見込み客の発見が容易。CEO・営業推進者の個人的なエンゲージメント成長が企業の認知向上につながることを訴求。長期契約による割引提案（年払い 3 ヶ月割引等）が有効。

## 市場リサーチ補足（market-research準拠）

### 市場規模推定
- **TAM（Total Addressable Market）**: BtoB企業のマーケティング・営業部門（日本国内約4,000～6,000社） × 月額2～3.5万円 ≈ 約96～252億円
- **SAM（Serviceable Addressable Market）**: LinkedIn・X利用のBtoB企業で営業組織20名以上 ≈ 約2,000～3,500社 ≈ 約48～147億円
- **SOM（Serviceable Obtainable Market）**: 初期3年で狙える市場 ≈ 200～500社 × 年単価36万円 ≈ 7.2～18億円

### 競合実態
- **Hootsuite**: BtoC向けで、LinkedIn機能が不十分。月額49～735ドル
- **Buffer**: SNS管理中心。BtoB向けのメール・分析機能が弱い。月額15～99ドル
- **HubSpot SNS機能**: 付随的な機能のみ。本格的なSNS運用には対応していない
- **本ツールの差別化**: 「BtoB向けSNS運用」「LinkedIn・X・note統合」「エンゲージメント→リード化の追跡」。廉価性で優位。

### 反証・リスク要因（このツールが売れない可能性）
- **SNS API仕様変更への対応負荷**: X（Twitter）、LinkedInの API仕様が頻繁に変更され、メンテナンスコストが予想以上に増加する可能性
- **BtoB企業のSNS活用の低迷**: 実は多くのBtoB企業は「SNSからのリード獲得効果」に疑問を持っており、継続的な投資を躊躇している可能性
- **CEO個人発信の限定性**: CEO個人のSNS発信効果は有限であり、スケーラビリティに限界がある
- **エンゲージメント→リード化の曖昧性**: LinkedIn訪問数増加 = リード品質向上という仮定が成立しない可能性
- **競合サービスの急速な台頭**: ChatGPTを用いたSNS投稿自動生成ツールが競合として参入する可能性

## 戦略評価（strategy-advisor準拠）

### 開発優先度
- **総合判定: 中**
- **理由**: BtoB企業のSNS運用需要は急増しており、競合が限定的。ただし複数SNS API連携の複雑性と、SNS仕様変更への継続対応が課題。「リード化」までを含めた効果測定が難しく、顧客の評価が不確実。中期フェーズ（初期6～12か月後）での投入が適切。

### 想定ROI（初期投資回収期間）
- **初期投資**: 開発費400万 + 営業・マーケティング初期費用150万 = 550万円
- **月額平均客単価**: 2.8万円（年間33.6万円）
- **想定初期採用速度**: 6か月で20社、12か月で50社
- **初期12か月MRR**: 20社×2.8万円=56万円 → 50社×2.8万円=140万円
- **回収期間**: 初期12か月で約105万円×12か月 = 1,260万円 → 5～6か月で回収
- **ROI指標**: 初期投資550万 / 3年LTV（50社×2.8万円×36か月）= 5,040万 → **ROI: 9.2倍（3年ベース）**

### 成功指標（KPI）
- **短期（6か月）**: ベータテスト企業10社からの「SNS運用工数削減」40%以上の実績報告
- **中期（12か月）**:
  - 有料顧客: 50社
  - MRR: 140万円
  - ユーザーNPS: 55以上
  - 顧客のSNS投稿頻度: 週3回以上に増加
- **長期（24か月）**:
  - 有料顧客: 120社
  - MRR: 336万円
  - 顧客がSNS経由でリード獲得した実績: 全顧客の40%以上
  - 顧客紹介による新規リード: 全新規リードの35%以上

### コンティンジェンシープラン（リスク対応）
**リスク1: 複数SNS API連携が複雑で、対応が遅れる場合**
- 対応: 初期段階ではLinkedInとX（Twitter）の2プラットフォームに限定。noteは次フェーズ以降で対応。

**リスク2: SNS API仕様変更により機能が動作しなくなる場合**
- 対応: 各SNS API公式ドキュメント常時監視体制構築。API変更への対応SLAを顧客に明示。

**リスク3: BtoB企業のSNS投資意向が期待値50%以下の場合**
- 対応: 「SNS運用効果測定」機能強化。顧客成功事例をケーススタディ化して訴求。CEO個人発信による企業認知向上のメリット強調。
