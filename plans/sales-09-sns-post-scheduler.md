# BtoB 向け SNS 投稿スケジューラー×効果測定ツール

## 概要
X（Twitter）・LinkedIn・note への投稿を一元管理し、スケジュール配信。エンゲージメント（いいね・リプライ・シェア）をトラッキング。BtoB 企業の SNS 運用に特化し、Hootsuite などの BtoC 向けツールとの差別化を実現。

## 目的・解決する課題
- 複数の SNS 運用が負担になり、一貫性のある投稿ができていない
- SNS 投稿の効果測定（エンゲージメント→リード化率）が不明確
- リード生成ツール（Hootsuite・Buffer等）は BtoC 向けで、BtoB に適していない

## ターゲット
BtoB 企業の情報発信・マーケティング部門。LinkedIn・X をビジネス発展に活用したいが、運用工数が足りていない層。CEO・営業推進者が個人発信も含めて統一管理したいニーズもある。

## 主要機能
- X・LinkedIn・note への投稿を一画面で管理・スケジュール配信
- 業界別・コンテンツタイプ別のテンプレート投稿提案機能
- 投稿のエンゲージメント（いいね・リプライ・シェア・クリック）をリアルタイム集計
- エンゲージメント の多い投稿パターンを AI が自動分析・提案
- リード化追跡（LinkedIn の プロフィール訪問数、招待リクエスト増加等）を計測
- チームメンバー用アカウントで、投稿スケジュール・承認フロー管理
- 業種別・フェーズ別のベストプラクティス投稿ライブラリ
- Google Analytics との連携で、SNS → Web サイト への流入追跡

## 競合との差別化
Hootsuite・Buffer は BtoC 向けで、LinkedIn 機能が不十分。HubSpot 等の MA ツールは SNS 機能が付随的。本ツールは「BtoB 向けの SNS 運用」に特化し、X・LinkedIn・note の最適化を実現。CEO・営業個人の発信を組織的に管理するユースケースも未開拓。

## 想定価格帯
初期費用: 25万円 / 月額: 2～3.5万円（メンバー数に応じた従量課金）

## 開発難易度
中 / Twitter API v2・LinkedIn API・note API との連携 + エンゲージメント計測ロジック + GA4 との連携で実装可能。各 SNS の API 仕様変更への対応がメンテナンスコスト。

## Agent所見
- **ディレクター**: BtoB 企業の「SNS による認知拡大 → リード獲得」というファネルを設計できる点が強み。ただし効果測定（SNS → 商談への帰属）が複雑なため、初期導入時の期待値調整が必須。
- **エンジニア**: 複数 SNS API の連携が実装のコア。各 API の認証・レート制限への対応は細かいが、対応難易度は高くない。GA4 統合で「SNS が本当にリード源になっているか」の可視化ができると差別化になる。
- **マーケター**: BtoB 企業の情報発信をスケーラブルにできる点が訴求のメイン。「個人の属人的な SNS 運用から組織的な情報発信へ」というスピン化が有効。業界別テンプレート投稿の充実度が競争力。
- **セールス**: LinkedIn ユーザー企業をターゲットにすることで、見込み客の発見が容易。CEO・営業推進者の個人的なエンゲージメント成長が企業の認知向上につながることを訴求。長期契約による割引提案（年払い 3 ヶ月割引等）が有効。
